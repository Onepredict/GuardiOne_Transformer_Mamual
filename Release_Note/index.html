Release Notes

v.1.1 - October 2020

Asset

Transformer Information List

  • Change contents of asset management: transformer name, register date, site, voltage(kV), capacity, manufacturer, manufacturing date, and operating date.

DGA Data Management

  • The DGA data list includes data acquisition date, H2, C2H2, C2H4, C2H6, CH4, CO, CO2, O2, N2, C3H8, H2O, TDCG, Acid, DGA analysis company, filtering date for both transformer DGA and OLTC DGA. The information such as data acquisition date, H2, C2H2, C2H4, C2H6, CH4, CO must be entered for the software to operate correctly.
  • The data input window will pop up when the user clicks the upload icon. Users can manually enter data in the data entry window.
  • Users can download all DGA data for a specific transformer. When you click the download icon, you can download data in csv file format.
